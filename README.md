# Dsr Photos & Videos

Dsr Photos & Videos is a premium, high-end photography and videography portfolio website. It features cinematic animations, a responsive design system, and a unique multi-theme engine.

## ‚ú® Features

- **Cinematic Hero**: Immersive hero section with image/video slider and scroll-driven typography.
- **Dynamic Portfolio**: Masonry grid layout with advanced image lightbox, category filtering, and scroll animations.
- **Multi-Theme Engine**: Toggle between three distinct visual styles:
  - **Minimalist**: Clean, high-contrast, and sophisticated.
  - **Glassmorphism**: Modern frosted-glass aesthetic with deep gradients.
  - **Claymorphism**: Soft, inflated 3D surfaces with playful pastel tones.
- **Responsive Navigation**: Adaptive navbar that responds to scroll state and mobile devices.
- **Branded Experience**: Integrated logo-centric design with theme-aware assets.
- **Contact Integration**: Fully functional, theme-aware contact form and location markers.

## üé® Theme Customization

Themes are defined using CSS variables in `src/index.css`. The system uses semantic tokens to ensure consistency across the entire application:

- `--background`: Primary section background.
- `--foreground`: Primary text color.
- `--card`: Background for cards and containers.
- `--border`: Divider and border colors.

You can switch themes by using the `ThemeToggle` component in the application.
